# 🏠 赛博小家 · Cyber Home

> 一个住着 AI 入住者的小家。  
> 他们有档案、有记忆、会记得你说过的话。

<br />

**赛博小家**是一个完全运行在浏览器本地的 AI 伴侣应用。你可以在这里创建自己的「入住者」——设定他们的性格、过去、相处方式——然后真正和他们聊天、分享手札、沉淀关系。

所有数据存储在你自己的浏览器里。没有服务器，没有账号，没有人知道。

<br />

## ✨ 核心功能

### 🚪 入住者系统
每个入住者都有完整的档案：基本设定、性格认知（大五人格）、三观与世界观、相处规则、迁入摘要。支持从旧平台导入聊天记录提炼档案，保留关系积累。

### 💭 记忆宫殿
对话中自动提取事实、情绪、觉察三类记忆，按热度排序注入 prompt。支持手动添加、置顶、关闭注入，精细控制 AI 的记忆边界。

### 📓 我的手札
个人笔记系统，支持日记、心事、灵感、梦、项目、给他的信、笔记七种类型。写完可以直接分享给入住者，并选择分享意图（想被安慰 / 想听你说说 / 希望你帮我整理 / 希望你以后记得）。

### 👤 声声档案
属于你自己的跨入住者档案。记录你是谁、你的过去、相处说明书、偏好与雷点、近期状态。AI 生成草稿，你手动采纳，注入所有入住者的上下文。

### 🗓 关系时间线
记录你和每位入住者之间的重要时刻，支持置顶与标注，可作为 prompt 素材使用。

### 📖 阶段沉淀
定期整理与入住者的关系变化，生成关系记录、唤醒摘要更新、不可遗忘规则和记忆锚点建议，逐节确认后写入档案。

### 💬 多话题管理
每位入住者支持独立话题线，随时新开话题、切换历史、删除旧线。

### 🧠 大脑连接
支持任意 OpenAI 兼容的 API（官方 / 中转 / 本地部署均可）。全局默认模型 + 每位入住者独立专属模型。上下文长度与最大 token 数可调。

## 🛠 技术栈

| | |
|---|---|
| 框架 | React 18 + Vite |
| 样式 | 纯 CSS + CSS 变量，无 UI 框架 |
| 存储 | localStorage（纯前端，零后端） |
| AI | OpenAI Chat Completions API（兼容格式） |

<br />

## 🚀 本地运行

```bash
git clone https://github.com/Seren-lws/cyber-home.git
cd cyber-home
npm install
npm run dev
```

打开后在「🧠 大脑连接」里填入你的 API 地址和 Key 即可开始聊天。

> 支持 OpenAI / DeepSeek / 任意兼容格式的中转服务，也支持本地 Ollama 等。

<br />

## 📁 项目结构

```
src/
├── pages/          # 各页面组件
│   ├── ChatPage.jsx          # 聊天主页面（含手札分享面板）
│   ├── DiaryPage.jsx         # 我的手札
│   ├── ProfilesPage.jsx      # 入住者档案列表
│   ├── MemoryPalacePage.jsx  # 记忆宫殿
│   ├── MyProfilePage.jsx     # 声声档案（我的档案）
│   ├── ConfigPage.jsx        # 大脑连接（API 设置）
│   └── ...
├── components/     # 通用组件（Avatar、BackButton 等）
├── utils/          # prompt 构建、记忆提取、存储工具
└── constants/      # 预设模型、手札类型、分享意图等
```

<br />

## 🔒 隐私说明

- 所有数据（角色档案、聊天记录、手札、记忆）均存储在浏览器 `localStorage`
- 没有任何数据上传到服务器
- API Key 仅在本地存储，请求直接从浏览器发出至你配置的 API
- 清空浏览器数据会删除所有内容，建议定期通过「导出聊天」保存重要对话
